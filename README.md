# Chinese-card

#### 介绍
PHP版本的AI汉语新解小卡片

#### 安装教程

1.  源码上传到服务器
2.  打开config.php设置模型信息
3.  配置完成后访问域名即可

#### 使用说明

1.  有时候会抽风生成失败
2.  建议使用32k模型
3.  演示：[https://card.ngnet.work/](https://card.ngnet.work/)
